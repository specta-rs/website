## What is Specta?

Specta is a powerful Rust library that enables you to export type information from Rust code and generate TypeScript bindings. It bridges the gap between Rust's strong type system and TypeScript, ensuring end-to-end type safety across your full stack applications.

### Key Features

- **Type-Safe Communication**: Automatically generate TypeScript types that match your Rust definitions perfectly
- **Zero Runtime Overhead**: All type generation happens at build time
- **Framework Agnostic**: Use Specta as a standalone library or integrate it with popular frameworks
- **Derive Macros**: Simple `#[derive(Type)]` annotations to export your Rust types
- **Flexible Export**: Support for functions, structs, enums, and more

### Project Goals

The primary goals of Specta are:

1. **Maintain Type Safety**: Ensure that changes in Rust code are immediately reflected in TypeScript, preventing runtime type errors
2. **Developer Experience**: Provide an intuitive API that makes cross-language type sharing effortless
3. **Performance**: Generate types efficiently without impacting build times
4. **Extensibility**: Allow the community to build powerful frameworks and tools on top of Specta
